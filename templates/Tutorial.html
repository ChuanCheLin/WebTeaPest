辨識流程與服務說明

Identification Process / Service Description

影像上傳

Image Uploading

使用手機拍照或透過電腦上傳田間病蟲害影像

病蟲害預測

Disease/Pest Prediction

檢視病蟲害預測結果，模型會預測受病害葉片位置並辨識病蟲害種類

結果查看

Information for Identification

展開結果可以查看病蟲害簡介，並透過連結至植保系統查看建議用藥

誤判回報

Misclassification Report

如果對判斷結果有疑問，可進行誤判回報， 該回報可以給專業人員處理結果並協助日後辨識模型的改進
